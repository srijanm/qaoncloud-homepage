# QAonCloud Homepage

Marketing homepage for QAonCloud.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- GSAP for animations

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Open `index.html` in your browser

## Project Structure

```
qaoncloud-homepage/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
├── node_modules/
├── package.json
└── README.md
```

## Development

This is a static site. Open `index.html` directly or use a local server.
